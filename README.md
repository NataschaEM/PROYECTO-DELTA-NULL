# PROYECTO-DELTA-NULL

PROYECTO DEL CURSO INTRODUCCIÓN A LA COMPUTACIÓN

Este proyecto consiste en la implementación de un scape room basado en el caso ficticio DELTA-NULL. Este caso ficticio está compuesto por 4 módulos.
Donde cada uno tenía un reto de corrección de código para decifrar mensajes encriptados.
La finalidad del proyecto es que las personas del equipo: Bryan Moreira, Natascha Espinoza, Diego Solera y Fabricio Campos. Resolvieran cada módulo obteniendo una contraseña para ingresar al archivo final cifrado.
Al completar los 4 módulos las contraseñas obtenidas de cada uno se unen para visualizar el mensaje principal encriptado.
El caso DELTA-NULL es un repositorio interno que presenta inconsistencias, versiones alteredas y registros fragmentados. Esto se da desde la desaparición de los programadores principales, las nuevas personas del equipo deben asumir el rol de programadores y recuperar los archivos. 
Durante una auditoria se descubre que el repositorio DELTA-NULL está compuesto por scripts, registros y archivos con inconcistencias temporales y múltiples versiones de una misma realidad.
El repositorio cuenta con 4 módulos dónde cada módulo tiene 4 puzzles y un archivo unlock, cada puzzle se debe ir resolviendo uno a uno y cada uno dará un contraseña que se usará para el unlock de ese módulo. Al finalizar los módulos y obtener sus respectivas contraseñas se utilizarán las mismas en el archivo merge_fragments es un sistema de construcción de ramas y este integra los fragmentos globales, que son las claves de cada módulo y brinda una contraseña final para el documento zip encriptado.
